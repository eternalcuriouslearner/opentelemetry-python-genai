OpenTelemetry Weaviate Client Instrumentation
=============================================

|pypi|

.. |pypi| image:: https://badge.fury.io/py/opentelemetry-instrumentation-genai-weaviate-client.svg
   :target: https://pypi.org/project/opentelemetry-instrumentation-genai-weaviate-client/

This library provides OpenTelemetry instrumentation for the
`Weaviate Python client <https://pypi.org/project/weaviate-client/>`_.

.. note::

   This package is under development and does not emit telemetry yet.

Installation
------------

::

    pip install opentelemetry-instrumentation-genai-weaviate-client

References
----------

* `OpenTelemetry Project <https://opentelemetry.io/>`_
* `Weaviate <https://weaviate.io/>`_
